# First Solution
> Approach:
>- ğŸ¯ Track Counters: track the maximum value for each sub-array.
> -ğŸ”„ Update Counters: For each element, compare with the previous:
	Increase Max: `max += nums[i]` if `nums[i - 1] < nums[i]` else `max = nums[i]`
	Reset if not increase.
>- ğŸ“ˆ Track Maximum: Update the global maximum after each step.
```java
class Solution {
    public int maxAscendingSum(int[] nums) {
        int maxGlobal = nums[0], max = nums[0]; 
        for (int i = 1; i < nums.length; ++i) {
            if (nums[i - 1] < nums[i]) {
                max += nums[i];
            }
            else {
                max = nums[i];
            }
            maxGlobal = Math.max(maxGlobal, max);
        }
        return maxGlobal;
    }
}
```
Time-Complexity: $O(n)$
Space-Complexity: $O(1)$
